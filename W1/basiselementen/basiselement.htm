Inleiding

In het vak static web leren we hoe je de visuele kant van een website moet programeren.
We gaan hier dus aan de slag met HTML & CSS

Wat is HTML?

HTML in een taal waar semantiek centraal staat. De naamgeving bepaald waarvoor een element gebruikt word.
Het is dus balengrijk om de betekenis van elk element te kennen.
HTML vorm dus een basisstructuur en verteld de browser waarvoor de geschreven centent meot dienen.

Wat is CSS?

CSS staat voor cascading style sheet. In deze taal wordt er door middel van verwijzingen naar html element, vorm gegeven.
CSS gebruiken we dus voor al de visuele aanpassingen.
